# Customer Handling

## Module 1: Customer Interaction Framework

Objective: কথা বলার একটা clear structure তৈরি করা

Topics:
•	Call/Chat flow:
    👉 Greeting → Listening → Clarifying → Solution → Closing 
•	First impression importance 
•	Professional opening line 

Practice:
•	Script reading + নিজের মতো করে বলা

## Module 2: Understanding Customer Need

Objective: আসল সমস্যা ধরতে পারা

Topics:
•	Right প্রশ্ন করা (open-ended / close-ended) 
•	Hidden problem identify 
•	Assumption avoid করা 

Practice:
•	“Customer কি বলছে vs আসলে কি সমস্যা” exercise

## Module 3: Clear & Confident Communication

Objective: সহজ ও পরিষ্কারভাবে explain করা

Topics:
•	Simple language use 
•	Jargon avoid করা 
•	Tone control (friendly but professional) 

Practice:
•	Complex problem → simple explanation 
•	Chat reply rewriting

## Module 4: Building Trust with Customer

Objective: কাস্টমারকে secure feel করানো

Topics:
•	Confidence show করা 
•	“I will help you” attitude 
•	Honest communication (false promise না দেয়া) 

Practice:
•	Trust-building বাক্য ব্যবহার 
•	ভুল হলে admit করা practice

## Module 5: Handling Angry Customers

Objective: conflict control করা

Topics:
•	কেন কাস্টমার রাগ করে 
•	De-escalation steps:
    👉 Listen → Acknowledge → Apologize → Solve 
•	What NOT to say 

Practice:
•	Role play (angry customer) 
•	ভুল response identify

## Module 6: Quick Problem Solving

Objective: দ্রুত solution দেয়া

Topics:
•	Problem identify → action 
•	Known issues vs unknown issues 
•	When to escalate 

Practice:
•	Time-bound সমস্যা solve 
•	Multiple case scenario

## Module 7: Multi-Channel Support (Call, Chat, Email)

Objective: সব মাধ্যমে effective হওয়া

Topics:
•	Call handling etiquette 
•	Chat speed + clarity 
•	Email format & professionalism 

Practice:
•	Same problem → call, chat, email তিনভাবে solve

## Module 8: Follow-up & Customer Retention

Objective: একবার solve না—relationship build

Topics:
•	Follow-up importance 
•	Feedback নেয়া 
•	সন্তুষ্টি নিশ্চিত করা 

Practice:
•	Follow-up message লেখা 
•	সন্তুষ্টি যাচাই script

## Module 9: Handling Complaints Properly

Objective: complaint কে opportunity বানানো

Topics:
•	Complaint receive করার সঠিক পদ্ধতি 
•	Documentation (note রাখা) 
•	Escalation process 

Practice:
•	Complaint case handling simulation

## Module 10: Live Simulation & Feedback

Objective: সবকিছু বাস্তবে প্রয়োগ

Activities:
•	Real-life scenario 
•	Live chat simulation 
•	Call role play 
•	Trainer feedback

Evaluation Criteria
•	Communication clarity 
•	Listening skill 
•	Emotional control 
•	Problem-solving speed 
•	Attitude
